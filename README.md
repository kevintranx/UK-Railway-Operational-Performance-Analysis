# UK-Railway-Operational-Performance-Analysis

## Project Overview

Analyzed over 31,000 UK railway journeys to identify patterns in service disruptions, passenger demand, delay severity, and route reliability. 

Built a dashboard using Microsoft Excel, pivot tables, and data visualizations to find insights and improve service reliability.

## Objective

Discover operational insights that could help improve overall rail service reliability.

## Key Insights

- Peak travel times occur between 6-8 AM and 4-6 PM, indicating strong commuter travel patterns. 
- Staff shortages produce the longest delays, at just over 51 minutes. 
- Peak train delays tend to increase after peak demand hours, suggesting possible traffic congestions.
- The Liverpool Lime Street → London Euston route experiences the highest number of disruptions with more than 800 cancelled or delayed trips.
